Pet Adoption App

This is a mobile application developed for personal and educational purposes, aimed at connecting users with pets available for adoption. Using this app, users can browse pets, view details, and find nearby adoption centers.

Purpose of the App

The primary goal of this app is to make pet adoption more accessible by providing a user-friendly platform to search for pets, save favorites, and connect with adoption centers. This app is designed for local development and testing with real-time pet data.

API Usage

The Pet Adoption App integrates with the Petfinder API to retrieve live data on pets available for adoption, including details such as breed, age, location, and more.
